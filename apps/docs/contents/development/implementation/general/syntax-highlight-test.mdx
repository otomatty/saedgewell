## Bash

```bash
# シェルスクリプトのサンプル
#!/bin/bash

# 変数の定義
NAME="World"
CURRENT_DIR=$(pwd)

# 条件分岐
if [ -d "./node_modules" ]; then
  echo "node_modules directory exists"
else
  echo "Installing dependencies..."
  npm install
fi

# ループ
for i in {1..5}; do
  echo "Count: $i"
done

# 関数
function greet() {
  echo "Hello, $1!"
}

# 関数の呼び出し
greet $NAME

# コマンドの実行結果を変数に格納
FILES=$(ls -la)
echo "Files in current directory: $FILES"

# 終了ステータス
exit 0
```

## 行のハイライト

```typescript {3-5}
// 特定の行をハイライト
function calculateTotal(items: Item[]): number {
  // この行はハイライトされます
  const total = items.reduce((sum, item) => sum + item.price, 0);
  // この行もハイライトされます
  return total;
}
```

## 単語のハイライト

```typescript /Item/ /price/
// 特定の単語をハイライト
interface Item {
  name: string;
  price: number;
  quantity: number;
}

function calculateTotal(items: Item[]): number {
  return items.reduce((sum, item) => sum + item.price * item.quantity, 0);
}
```
